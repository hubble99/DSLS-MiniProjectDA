# DSLS-MiniProjectDA
Proyek ini merupakan MiniProject Data Analyst yang bertujuan untuk membuat sales report berupa dashboard dengan menggunakan Power BI. Analisis ini menggunakan data dari perusahaan Northwind Traders.

## Tujuan Proyek

Dashboard penjualan ini memiliki beberapa tujuan utama:
1. Menyajikan laporan penjualan dengan menggunakan visualisasi yang jelas dan mudah dimengerti.
2. Membantu perusahaan Northwind Traders untuk memantau kinerja bisnis secara keseluruhan dan melihat tren penjualan yang ada.
3. Memungkinkan perusahaan untuk membuat keputusan bisnis yang lebih baik dan merancang strategi yang lebih efektif.
4. Identifikasi masalah atau peluang bisnis dengan cepat, memungkinkan perusahaan untuk mengambil tindakan yang diperlukan.
